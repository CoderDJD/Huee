# Contributing :

PLEASE READ CODE_OF_CONDUCT.md BEFORE GOING ANY FURTHER.
**You** can contribute to this repository by any of the following means :

- Becoming a maintainer.
- [Creating an issue](https://docs.github.com/en/github/managing-your-work-on-github/creating-an-issue)
  - Proposing new features
- [Creating a pull request](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request#creating-the-pull-request)
  - Submitting a fix, to an issue.

**Contributing Code** :<br>
**_Note :_** Want to be a maintainer? Please DM me at [Twitter](https://twitter.com/home) or [Discord](https://discord.com/app) @CoderDJD.

- Fork the repository, clone it to your local machine, and run these commands in the root directory of the project.

  ```sh
  npm install # yarn for yarn users
  cd client/
  npm install
  yarn # compulsory, make sure you have yarn installed.
  ```

- Make the necessary changes. And from the bottom of our heart, thanks. To stage, commit and push changes, run the following commands.

  ```sh
  git add *
  npm run commit
  git push
  ```

and open a new [pull request](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request#creating-the-pull-request) to let us know that you have a change.
