**Contributing Code** :

**_Note :_** Want to be a maintainer? Please email me [here](mailto:natechnologies2007@gmail.com) or DM me at Twitter(@CoderDJD) or Discord(@CoderDJD).
The changes happen through pull requests.

- Fork the repository, clone it to your local machine and then :

In the root directory of the project, run :

```
npm install
cd ./vue && npm install
```

- Stage the files to be commited, in root directory, run :

```
git add *
```

**_Note :_** The commit messages follow the [conventional committs syntax](https://www.conventionalcommits.org/en/v1.0.0-beta.2/).

- Make sure you're in the root directory of the project. Then run

```sh
npm run commit
```

to open up an interactive terminal prompt which will commit the changes.

- Push all changes,

```sh
git push
```

and open a new [pull request](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request#creating-the-pull-request) to let us know that you have a change.
